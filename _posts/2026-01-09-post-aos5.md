>   **디바이스 로그 관련 상세 내용 및 취약점** <br>




테스트
안드로이드 디바이스 로그(logcat 등)에 중요 정보가 저장되는지 여부를 확인한다.ㅁㄴㅇㅁㄴㅇ 





adb shell dumpsys activity activities | findstr "패키지명"
하면 지금 켜져있는 액티비티 알수있음
adb shell am start -n 패키지명/액티비티명 하면 액티비티 강제실행가능

[오전 10:42] logcat --pid [pid] | grep "원하는문자열" 하면 디버그 로그 노출 찾을수있고


일단 메뉴누르고 그거에 대한 엑티비티 찾을떄 써




디버깅관련
<application android:debuggable="true" /> ->2020년 이후 거의없음

adb logcat | findstr reserve
adb shell dumpsys package <패키지명>
adb shell dumpsys package kr.co.Outback | grep -i debuggable
adb shell dumpsys package kr.co.Outback |findstr /i debug (윈도우)
